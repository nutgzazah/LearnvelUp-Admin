**LearnvelUp Admin** is the back-office management dashboard for the LearnvelUp mobile application. It is built to manage users, content, and system settings, utilizing the same database and design system as the mobile app.

## ‚ú® Key Features

- **Authentication:** Secure admin login via Supabase Auth.
- **Responsive Design:** Fully responsive layout for Desktop.
- **Dark Mode Support:** Seamless Light/Dark mode switching using `next-themes`.
- **Custom Design System:**
  - Integrated **K2D** typeface (Local Font) to match the mobile app's branding.
  - Unified color palette (Primary/Secondary) via CSS variables.
